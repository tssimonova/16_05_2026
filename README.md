# Свадебное приглашение - Бильярдная тематика

Интерактивное сайт-приглашение на свадьбу с бильярдной тематикой.

## Как запустить локально

### Способ 1: Использование Python (рекомендуется)
```bash
python3 -m http.server 8000
```
Затем откройте в браузере: http://localhost:8000

### Способ 2: Использование Node.js
```bash
npx serve .
```
Затем откройте в браузере: http://localhost:3000

### Способ 3: Использование PHP
```bash
php -S localhost:8000
```
Затем откройте в браузере: http://localhost:8000

## Как использовать

1. Откройте сайт на мобильном устройстве или в десктопном браузере
2. Сделайте свайп вниз (или клик и потяните вниз на десктопе)
3. Наслаждайтесь анимацией удара кием и разлета шаров
4. После анимации появится приглашение с вашей фотографией

## Особенности

- Полностью адаптивный дизайн для мобильных устройств
- Реалистичная анимация бильярдных шаров и кия
- Фон в виде бильярдного сукна
- Плавные переходы и анимации
- Поддержка touch-событий для мобильных устройств

## Структура файлов

- `index.html` - основная HTML структура
- `style.css` - стили с реалистичными эффектами
- `script.js` - интерактивность и анимации
- `1.jpg` - ваша свадебная фотография
- `README.md` - этот файл
